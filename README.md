## requirements:
- npm
- java

## steps to install:
1. Clone repository
2. ``` npm install ```
3. ``` npx playwright install --with-deps ```

## steps to launch:
```
npx playwright test
```

## steps to creating the report:
```
npx allure serve allure-results
```
